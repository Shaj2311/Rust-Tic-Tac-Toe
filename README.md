# Rust-Tic-Tac-Toe
A basic command-line based implementation of Tic Tac Toe, written in Rust.

# How to Run
1. Clone this repo to your machine
```
git clone https://github.com/Shaj2311/Rust-Tic-Tac-Toe.git
```
2. Navigate to the repo
```
cd ./Rust-Tic-Tac-Toe
```
3. Build and execute using Cargo
```
cargo run
```

# Code Overview
This game is implemented inside a single .rs file `src/main.rs`. It may be poorly written (it's my first Rust project) but does not use any dependencies and relies solely on the `std` crate.
